# Data

Datasets for the EARTHLAB 2026 practicals.

- **argo_current_reconstruction/** — Argo float velocities + WOA temperature/salinity for the ANN regression practical
- **sea_ice_dataset/** — Arctic sea ice satellite images (128×128 grayscale) for the CNN classification practical
- **Glac_Melt.csv** — daily glacier melt vs air temperature, Spitsbergen 1999 (courtesy of Prof. Gareth Rees), for the Bayesian inference practical

See the README inside each dataset folder for full documentation.
